# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.4] - 2026-04-13
### Added
- Added presentation images and examples to `README.md`.
### Changed
- Standardized project namespace to `alama777`.
- Fixed extension repository URL configuration in package constraints.
- Enhanced tool documentation and parameter descriptions.
- Stabilized internal SSE and CDP connectivity handling.

## [0.0.1] - 2026-04-13
### Added
- Initial experimental release of Antigravity MCP Server.
- Tool: `start_new_chat` with background session support.
- Tool: `send_prompt` for non-blocking message injection.
- Tool: `get_active_chat` for DOM-based scraping via CDP.
- `bin/stdio-proxy.mjs` bridge for external AI clients.
- Comprehensive `README.md` and `INSTALL.md` documentation.
- Zero-config automatic injection for internal Agent Manager.
